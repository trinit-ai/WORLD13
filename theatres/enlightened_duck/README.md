# Enlightened Duck Theatre

**Pack:** enlightened_duck (password: rcv2026)
**Agents:** 10 — one per primary archetype
**Sessions:** 1 per agent (one full Duck reading each)
**Purpose:** The same oracle encounters 10 different human situations.
The divergence in what each agent receives, and what it means for their arc,
is the output.

## The Question

The Enlightened Duck asks three questions. The same three questions to everyone.
What changes is who is asking — and therefore what the questions surface.

A CEO in CRS phase approaches the same silence as a Witness in RES phase.
The Duck addresses both. The readings diverge completely.

## What the Analysis Produces

- **Liberation path alignment**: Which LP does each reading point toward?
- **K(x) delta projection**: What is the estimated karmic impact of receiving this reading?
- **Question divergence map**: How differently did the 10 agents interpret Q1, Q2, Q3?
- **Life path fork summary**: For each agent — the shape of the work that remains.

## Running the Theatre

```bash
make duck
```
