# User-Forked Packs

This directory contains personal pack forks created via `/fork`.

- Each fork is a subdirectory: `protocols/user/{pack_id}/`
- Fork packs contain: MANIFEST.md, header.yaml, master.md, manifest.json
- Fork packs are loadable with `/load {pack_id}` just like library packs
- Forked packs are NOT committed to the shared library by default
- Create a fork: `/fork <source_pack_id>`
- List your forks: `/forks`
